Image Search Abstraction Layer Microservice

Search Instructions

https://consolelogreece-isalms.glitch.me/api/imagesearch/<YOUR-SEARCH>

example: https://consolelogreece-isalms.glitch.me/api/imagesearch/fluffy kittens

for pagination/offset, simply add "?offset=<OFFSET>"

example: https://consolelogreece-isalms.glitch.me/api/imagesearch/fluffy kittens?offset=2
